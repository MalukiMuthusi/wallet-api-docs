get /wallets/{wallet_id}/balance
Returns a wallet's balance amount (walletsWalletIdBalanceGet)
Returns the balance of the wallet that has the provided id

Path parameters

wallet_id (required)
Path Parameter — Refers to a wallet by its id format: int32

Return type

Balance

Example data

Content-Type: application/json
""

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

The returned balance amount Balance

400

Invalid wallet id provided BasicError

404

Wallet not found BasicError

500

Server encountered error when processing your request. This a server error. When in debugging session check the response to see what happened to the server BasicError

501

The operation has not been implemented yet. BasicError
